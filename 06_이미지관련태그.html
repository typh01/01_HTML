img 태그

    - 웹 문서에서 이미지를 삽입하는 용도의 태그

    [속성]
    (속성 : 시작태그에 작성하는 내용)

    1. src : 삽입할 이미지의 경로를 작성하는 속성
            (파일 위치, 인터넷 주소(url))
    2. width / height : 너비 / 높이를 지정하는 속성
            (단위 : px, %, em, rem 등이 존재)
    3. alt : 이미지 설명을 작성하는 속성
            1) 지정된 경로에 이미지가 없을 때 보여줄 내용 작성
            2) 웹 접근성(이미지 설명)

img 태그 기본 사용법

이미지 크기 지정 (width/height)

고정 크기 단위

    - 화면의 크기가 변해도 항상 고정된 크기를 지님
    - 단위 px

가변 크기 단위

    - 브라우저 크기 또는 지정된 영역의 크기에 대한 
      상대적인 크기 단위

    - 단위 : %, em, rem 등

고정된 크기의 영역 내부에 가변 크기 이미지
