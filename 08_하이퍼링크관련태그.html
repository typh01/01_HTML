[용어 정리]
    Hyper : 초월
    Link : 연결
    Hyper Link : 현재 문서를 넘어서 다른 문서와 연결
    
    Hyper Text : 클릭하면 연결된 다른 문서로 넘어가는 글자(문자열)

    HTML(Hyper Text Markup Language)
    : 다른 문서와 연결 될 수 있는 글자를 작성할 수 있고
      태그 형태를 해석해서 화면에 출력하는 언어

    HTTP(Hyper Text Transfer Protocol)
    : 인터넷을 Hyper Text 가 작성된 문서(HTML)를
      서로 전송할 때의 약속(규약)

a(Anchor, 연결) 태그

    - 현재 문서(HTML)와 다른 문서(HTML)를 연결 시켜
      클릭 시 이동하게 만드는 태그

    - a 태그로 감싸진 글/내용/이미지 == Hyper Text

    - Hyper Text는 다른 문서오 구분되는 HTML 만의 핵심 기능!!
      (이었다가... 요즘은 다들 잘 쓰고 있음...)
